# SDX Theme for VS Code

## Match with System

```jsonc
// .vscode/setting.json
{
  "window.autoDetectColorScheme": true,
  "workbench.preferredLightColorTheme": "SDX Light",
  "workbench.preferredDarkColorTheme": "SDX Dark"
}
```
